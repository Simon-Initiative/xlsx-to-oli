  1. Create a Google Doc that contains your desired workbook contents.
  2. Create a Google Docs API Key. See https://developers.google.com/docs/api/quickstart/js for simple instructions to enable the API for use in your browser.
  3. After you create your API key, edit the key at https://console.developers.google.com Click the Credentials tab, then click the pencil icon (edit) for the Oath 2.0 Web Application client. From the next page, you need to add the entry http://stormwind.oli.cmu.edu:9001 to the Authorized JavaScript origins. You will likely be prompted to add cmu.edu to your authorized domains list in order to complete this step.
    This step is optional and only necessary if you want to use this utility as a hosted solution from stormwind. Alternatively, you can check out this code and run it locally from your localhost.
  4. Paste your client id, API key, and your Google Docs document id into the form below and click Submit. The result will be a zip file containing the workbook xml and any supporting image files.

Supported Elements

  • Paragraphs
  • Markup including bold and italic
  • Hyperlinks
  • Tables
  • Embedded images
  • Ordered and bulleted lists
  • Nested lists

Custom Element Support

This tool supports the importing of non-native Google Docs elements through a CustomElement table that the author of the doc inserts.

YouTube Video
CustomElement youtube
src the video id
caption optional - the video caption
height optional - height in pixel
width optional - width in pixels
Formative assessment
CustomElement formative
idref the resource id of the formative assessment
Summative assessment
CustomElement summative
idref the resource id of the formative assessment

To see a sample doc that demonstrates how to include custom elements, checkout out: https://docs.google.com/document/d/1URR7Ii4LFQwhHllqYtV3sHaU7tQMeIUzG0iU6qm27Z0/edit?usp=sharing.
